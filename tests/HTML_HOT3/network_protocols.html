Three Network Protocols:

HTTP
One of the most familiar protocols, hyper text transfer protocol (HTTP) is often referred to as the protocol of the internet. HTTP is an application layer protocol that allows the browser and server to communicate.
SNMP
Is used to monitor and manage network devices.
SSL
This is a protocol mainly used for making sure a the internet connection is secure. while also providing protection for sensitive data.
